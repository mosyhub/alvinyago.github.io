About the Project

This rotoscope animation project involves frame-by-frame editing to isolate and stylize motion. It was done entirely using Adobe After Effects 2023 tools, combining manual brush work, keyframes, effects, and layering techniques.

Tools Used

RotoBrush Tool

Allows you to separate a subject from its background using painted strokes. Essential for clean cutouts frame-by-frame.

Pen Tool

Used to draw precise masks and outlines, especially where the RotoBrush wasn't enough.

Effects & Presets Panel

Used tools like Fill, Glow, and Tint for visual appeal. Great for fast experimentation and style.

Transform Properties + Stopwatch

Used keyframes to animate movement: zooming, fading, and panning made the shot more dynamic.

Freeze Feature

Locks the roto selection so After Effects doesn't recalculate it. Helps maintain consistency throughout.
